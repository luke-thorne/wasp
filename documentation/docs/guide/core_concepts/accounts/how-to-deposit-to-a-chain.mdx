# How to Deposit to a Chain

Any assets attached to an on-ledger request are automatically deposited to the sender's L2 account before actually executing the request.
So, in order to deposit tokens, just send _any_ on-ledger request with the tokens attached.

A commonly needed operation is to just deposit some funds and do nothing else.
The `deposit` entry point of the [`accounts` core contract](../core_contracts/accounts) is a no-op function that serves this purpose.

:::note
* All requests are charged a gas fee, so the L2 account may end up having fewer tokens than the amount sent.
* The IOTA L1 transaction needs to have a minimum amount of tokens attached for
  storage deposit. If the amount transferred is less than this minimum amount
  it will fail.
:::
